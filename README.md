# Sistema-de-Cadastro-e-Gerenciamento-de-Livros-da-Biblioteca-Escolar
A escola identificou a necessidade de organizar melhor o acervo de livros da biblioteca, atualmente registrado em planilhas e anotações manuais. Por isso, os alunos devem desenvolver uma aplicação web simples para simular um Sistema de Gerenciamento de Livros, permitindo inserir, visualizar, atualizar e excluir registros (CRUD).
